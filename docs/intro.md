\section{本書の背景}
中学高校時代から数学テキストを読み漁ってきた数学徒には, 代数・幾何・解析の各分野を横断して横たわる数学的構造は自ずと頭の中にイメージできているのだろうと思う. 一方で大学1年生になり初めて佐武一郎著「線型代数学」を触れるような多くの一般的理工系学生にとってはいきなり「一般に集合$K$において加法及び乗法と称する二つの演算が定義されていて, それらに関して以下の公理が成立するとき, $K$を一つの体という」, 「$n$次正方行列全体からなる集合は体にはならない」なる文章に遭遇すると面食らうのではないか?

体とは何かを理解をすべく「群・環・体」に手をだし, 芋づる式に「集合・位相」まで手をつけ始めるとそれで半年を消化してしまい, 本来目的である「線型代数学」習得にたどり着けないことになるのが関の山である.

同様にルベーグ積分や確率を習う段になると有限加法族, $\sigma$加法的集合族, ボレル集合族なる集合族に遭遇する.小針 晛宏著「確率・統計入門」では「(ボレル集合は)どんな集合かイメージが把えにくいかも知れないが, (中略)ボレルでない集合というのは大変に病理的で専門的なもので, 余り役立たない」とある.ある位相空間全体の開集合全体によって生成される$\sigma$加法族をボレル集合族といい, このボレル集合族に属する集合をボレル集合というのだが, 果たして解りにくい.

数学徒以外の一般的な理工系学生によって要領の良いアプローチは, 集合から始めて, 群, 環, 体，加群，多元環からネーター環，作用素環，ホモロジー代数まで速習し, その数学的構造のイメージをつけることのように考える.

本書の目的は数学徒以外の一般的な理工系学生向けに, 集合から出発し, 線型空間，位相空間，可測空間までについてざっと駆け上ることで, 現代数学の基礎となる数学的構造を俯瞰，見通しをよくすることにある.「駆け上がる」ために,本書では各数学的構造での例を主に以下の4つ
\begin{enumerate}
    \item 要素数が2つだけの集合,
    \item 自然数，整数, 有理数，実数，複素数全体からなる集合,
    \item $n$次正方行列全体からなる集合,
    \item 実数の区間$[0,1]$上で定義された実数値関数全体からなる集合.
\end{enumerate}
に絞り，可能な限り上記の順序となるように例示した．

\section{本書の構成}
本書の大きな展開は，数学の歴史的な構成の流れを踏まえている．即ち
\begin{itemize}
    \item 具体的な数学的構造の構成
    \item 圏論による数学的構造の再構成
    \item ホモロジー代数やK理論，非可換幾何学の展開
\end{itemize}
の3段階を踏まえている．各章節の構造は上記3段階には正確に合致しないが，本書は大きく以下のパートで構成されている．
\begin{itemize}
    \item 集合，写像，関係
    \item 数学的構造
    \begin{itemize}
        \item 代数的構造
        \item 順序的構造
        \item 位相的構造
    \end{itemize}
    \item 数学的構造間の関係
    \item 圏論入門
    \item ホモロジー代数入門
\end{itemize}
本書では以下を意識した．
\begin{itemize}
    \item 集合からネーター環，作用素環，ホモロジー代数など具体的な数学的構造を駆け上がること
    \item 駆け上がった頂きから普遍性，自由など圏論の視点から俯瞰できること
    \item 代数，順序，位相構造をカバーし，それら関係を見渡すこと 
\end{itemize}

「駆け上がる」ために数学的対象の細かな性質(多分に重要な性質も含まれる)の記載は飛ばす．集合から出発し, 線型空間，位相空間，可測空間」までなので，線型空間以降の内積空間，プレヒルベルト空間，ノルム空間，バナッハ空間，ヒルベルト空間についても触れない．線型空間以降の空間に暮らしている物理学徒や理工系学生が線型空間と集合の間にある隙間を速習することの一助になれば幸いである．

\section{表記}
集合を$S,T,U,V$, 未知の集合を$X,Y$, 集合の要素を$p,q,r,u,v$，べき集合を$2^S$, 集合系を$\mathfrak{U}, \mathfrak{B}, \mathfrak{M}, \mathfrak{O}, \mathfrak{A}$, 写像を$f, g, h$で表す．他，群に$G$, 加群に$M$, 環に$R$, 体に$F$，測度を与える集合関数に$\mu$を使う．二項演算子には$\circ, +, \times, \cdot$を使う．空集合は$\varnothing$を使い，普遍集合に$U$を用いる．集合系にドイツ文字を用い，圏論の用語に花文字を用いる．数学用語として一般化していない本書独自の造語は二重引用符をつけて用いる．

始集合$S$の元$s$から終集合$T$の元$t$への写像$f$は
\begin{align}
    f: S &\rightarrow T,\\
    s &\mapsto t
\end{align}
と2行に分けて書かかずとも理解できる場合には, $f: S \rightarrow T, s\mapsto t$と1行で記す.
