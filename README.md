# Category Theory Notes

This repository contains structured notes on category theory and related areas of algebra and homological algebra.

The content is published as a documentation site:

https://seikato.github.io/categorytheory/

---

## Overview

These notes develop a conceptual path from basic algebraic structures to advanced categorical constructions.

We aim to build intuition step-by-step through the following progression:

- Complexes  
- Chain complexes  
- Homology  
- Homological algebra  
- Standard constructions (e.g. comonads)  
- Co-Kleisli categories  
- Cartesian differential categories  

---

## Learning Path

The material is organized as a natural progression:

1. Complexes  
2. Chain complexes  
3. Homology theory  
4. Homological algebra  
5. Categorical constructions (monads / comonads)  
6. Co-Kleisli categories  
7. Cartesian differential categories  

---

## Purpose

The goal of this repository is to provide:

- A structured introduction to category theory  
- Connections between algebra and categorical structures  
- A readable path from classical homological algebra to modern categorical frameworks
